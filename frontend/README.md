# Notes App Frontend (Minimal)

Open `frontend/index.html` in a browser to interact with the Notes backend.

## How to run

1. Start the backend:
   ```bash
   cd notes-app
   npm install
   npm start
   ```
2. Open:
   - `notes-app/frontend/index.html`

## Usage

- Set **API Base URL**.
- **Register** then **Login** to get a JWT token.
- Paste JWT into the **Token** textarea.
- Use CRUD, share, pin/unpin, and search.
